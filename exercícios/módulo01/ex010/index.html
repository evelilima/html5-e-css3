Usando links externos

Para aparecer nos buscadores do Google estude sobre SEO (Search Engine Optimization).

Usando links externos

Envelopa a frase que você quer transformar em link e use a tag <a> vai surgir um parâmetro href="" entre as aspas vc coloca o endereço do site.

⇒ para o repositório do Prof. Gustavo guanabara no GitHub

⇒ Acesse Também a aula em vídeo número 10 no YouTube

Dessa forma você sobrepõe o site externo ao seu, é muito provável que a pessoa não retorne mais ao seu site. o que podemos fazer?
quando for um link externo sempre use o parâmetro target="_blank" rel(relativo a) ="external" depois da url. Assim abrirá o site em outra aba. Isso se aplica em links que levam a outros sites que não os seus

            
                <a href="https://github.com/gustavoguanabara/html-css/blob 
 /master/aulas-pdf/10%20-%20Liga%C3%A7%C3%B5es%20em%20toda%20parte.pdf" 
 target="_blank" rel="external">repositório do Prof. Gustavo guanabara 
 no GitHub</a>

Usando links internos

Links internos farão relação com outra página que está dentro do nosso servidor e diz respeito a nossa própria página. Para isso temos que ter outras páginas.
Temos dois tipos de link interno.
temos o index geralmento por onde o site é aberto e lá podemos criar novo arquivo com o nome pagoo2.html, por exemplo, essa será minha página 2.

Se você quiser ver a minha segunda página, clique aqui.

Use tag <a></a> e entre as aspas aperte ctrl + espaço que aparecerá todas as páginas que você tiver feito e é só escolher. Mas para que isso funcione as páginas criadas tem que estar na mesma pasta index

Não existe o parametro rel="external" nem "internal" mas podemos usar rel="next" (próxima) no index e "prev" (anterior) na pag002. Isso será útil para o mecanismo de busca saber qual é o seu sistema de navegação. Você não vai deixar o google decidir qual é a próxima página você ja está dizendo pra ele.

Você também pode acessar minha terceira página

podemos colocar um target indicando que a página que vamos abrir é para a própria página para não ficar abrindo várias abas pois os links são internos então não há necessidade. para isso use target="_self". Self significa na própria aba.
