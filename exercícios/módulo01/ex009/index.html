Trabalhando com Listas

Listas Ordenadas

<ol> </ol> significa Lista Ordenadas
<li></li> refere- se a cada item da lista.

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
        
            <ol>

                <li>Acordar</li>

                <li>Ligar para o João</li>

                <li>Tomar café</li>

                <li>Escovar os dentes</li>

            </ol>

O </li> é opcional

Podemos usar em <ol> o parâmetro type= que vai fornecer cinco opções (1 - A - a - I - i) 1 é a lista númerada que começa por padrão em 1, se você quiser que começe em 4 por exemplo, precisa incluir start="4".

        
            <ol type="">
                <li>Acordar</li>
                <li>Ligar para o João</li>
                <li>Tomar café</li>
                <li>Escovar os dentes</li>
            </ol>
  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
        
            <ol type="1" start="4">
                <li>Acordar</li>
                <li>Ligar para o João</li>
                <li>Tomar café</li>
                <li>Escovar os dentes</li>
            <ol>

Listas de Definições

É como se fosse um dicionário, você coloca o termo e a definição dele.

<dt>: Termo
<dd>: Descrição

HTML
Linguagem de Marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.
        
            <dt>HTML</dt>
            <dd>Linguagem de Marcação para a criação do conteúdo de um site.</dd>
            <dt>CSS</dt>
            <dd>Linguagem de marcação para a criação do design de um site.</dd>mark style="background-color: rgb(43, 226, 104);"
            <dt>JavaScript</dt>
            <dd>Linguagem de programação para a criação de interatividade de um site.</dd>
