Exemplo de hover

        
            <style>
                div{
                    text-align: left;
                }
                div > p {
                    display: none;
                }
                div:hover > p {
                    display: block;
                    color: black;
                    background-color: #6D96BD;
                    width: 200px;
                }
                div:hover {
                    color: #475C70;
                }
            </style>
        

        
            <body>
                <p>Passe o mause sobre o texto abaixo:</p>
                <div>
                    Passe o mause aqui!
                    <p>Texto escondido...</p>
                </div>
            </body>
