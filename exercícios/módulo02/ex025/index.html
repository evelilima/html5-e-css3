Váriáveis personalizadas com CSS

Cadastrar os esquemas de cores e fontes do nosso site vai facilitar na hora de efetuar mudanças estéticas.

Basta criar uma pseudo-class chamada :root. As váriáveis personalizadas em CSS devem ter seus nomes iniciados com dois traços obrigatóriamente.

        
            :root{
                --cor0: #c5ebd6;
                --cor1: #83e1ad;
                --cor2: #3ddc84;
                --cor3: #2fa866;
                --cor4: #1a5c37;
                --cor5: #063d1e;
                --font-padrao: Arial, Helvetica, sans-serif;
                --font-destaque: 'Bebas Neue', 'cursive';
                --font-android: 'Android', sans-serif;
            }

Uso do seletor * em CSS

Ele basicamente aplica uma configuração padrão para TODOS os elementos do código HTML ao qual o estilo está sendo aplicado.

        
            *{
                margin: 0px;
                padding: 0px;
            }
