Aula 26 _ Sombras nas caixas

Sombras nas caixas

As sombras são muito úteis para dar volume, para dar a sensação de que as caixas estão ali realmente. 

        
            <style>
                box-shadow: 1px(deslocamento horizontal) 
                    1px(vertical) 1px(espalhamento) black(sombra);
            </style>

Aula 27 _ Bordas decoradas

vértices arredondados

Podemos arredondar os vértices usando a declaração border-radius. border-radius: 10px

Se for necessário, podemos insicar quatro medidas diferentes, uma para cada vértice: border-radius: 10px 20px 30px 40px

também é possível indicar apenas dois valores, o que vai agir em vértices intercaladas, partindo do canto superior esquerdo: border-radius: 10px 30px

Com esse código abaixo podemos fazer uma bola: 

        
            <style>
                div#bola {
                    height: 100px;
                    width: 100px;
                    margin: 30px;
                    background-color: white;
                    border-radius: 50%;
                }
            <style>

Pode usar porcentagem no lugar de pixels.
